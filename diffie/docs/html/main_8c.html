This is a command-line program which implements Diffie/Hellman/Merkle in three possible ways:

  1. A local program which simulates a conversation between a client and a server, generating the needed packets and simulating the traversal of a network to generate the shared secrets.
  2. A server: this program can stand up an impromptu server which receives requests from a client to connect using a rudimentary network protocol used here for demonstration purposes.
  3. A client: this program can be configured to connect to a server instance and will initiate the conversation with the server in order to establish the shared secret.
